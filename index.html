MoleEngine

A fully featured 2D game engine written in C++ with game state and asset management, physics simulation, and various animation features. It is very early in development, but already features game state management and some collision detection capabilities.

2017

I Wanna Conquer the Chaos Cave

A precision platformer in the subgenre known as I Wanna Be the Guy fangames. Based heavily on the mechanics of I Wanna Be the Guy, the game contains difficult spike-dodging platforming and a chaotic avoidance boss fight. My core design challenge was to create a boss that has randomness in every attack yet is always possible and fair, unlike most bosses in the genre. Made in GameMaker: Studio.

No Man's Canvas

No Man's Canvas is a clone of Reddit's r/Place. It's a sandbox where players use a limited number of pixels to draw on a shared canvas in real time. Pixels refresh with time, and drawing more increases the number of pixels you can hold at a time. I worked on the ReactJS-based front-end. The back-end is written in Swift and runs on my friend's personal server.

The Division of Sweden

The Division of Sweden is a virtual reality passport inpection game inspired by Papers, Please. You play as a border agent on the volatile border of East Sweden and West Sweden, tasked with keeping illegitimate immigrants on their side of the border. You must make sure all the information matches on all of the travellers' documents. The game was made in Unity and requires an Oculus Rift VR headset and a Leap Motion hand sensor to play.

2012-2015

Flash Game Engine

I never actually made anything with this because I began my military service before I managed to get everything I needed working, but this was my first attempt at creating complex interactions between game objects. It featured collision detection between arbitrary convex polygons, circles and line segments, optimized with a simple grid-based spatial partitioning system, and basic (buggy) physics simulation on collisions. It also had the beginnings of an AI movement system, but all of this ended up very unfinished.

Minecraft: Ocean Adventures Mod

My most popular release to date with over 20,000 downloads, the Ocean Adventures mod for Minecraft started in 2012 as an effort to tweak the Minecraft world generator to create more interesting oceans. Through adjusting the game's existing biome system it generates a world entirely covered in an ocean and dotted with various types of islands. The mod grew over time to also add a variety of items and entities, such as a speedboat for quickly traversing the ocean, a new farmable crop and an ore that only appears on the seafloor. Development has been discontinued so the mod is not compatible with the latest versions of Minecraft.
